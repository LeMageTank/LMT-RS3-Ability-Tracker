Build instructions:
	1. Install PyInstaller.
	2. Run pyinstaller actiontracker.spec in the project root directory.
	3. The built project will be located in dist\LMT's Ability Tracker

Required dependencies for running or building from source (all listed in requirements.txt for install with pip):
	1. pynput
	2. pyautogui
	3. Pillow
	4. numpy
